# Website Dra. Inês Sapinho

Website estático multi-página, em português de Portugal, para a Dra. Inês Sapinho.

## Comandos

```bash
npm run build
npm run serve
```

O build gera a pasta `public/`, que é a pasta configurada para deploy na Vercel.

O conteúdo principal está centralizado em `src/data/profile.ts`.  
Os componentes de renderização estão em `src/components/site.ts`.

## Páginas

- `index.html`
- `sobre.html`
- `areas.html`
- `congressos-eventos.html`
- `livro.html`
- `publicacoes.html`
- `destaques.html`
- `marcacao.html`

## Antes de publicar

Validar fotografia oficial, autorização da capa do livro, link oficial de LinkedIn, cargos atuais, publicações, links de media/podcast, FAQ clínica, páginas legais, domínio final e autorização para exibir certificados/documentos.
