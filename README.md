# Website Dra. Inês Sapinho

Website estático, em português de Portugal, para a Dra. Inês Sapinho.

## Comandos

```bash
npm run build
npm run serve
```

O build gera a pasta `public/`, que é a pasta configurada para deploy na Vercel.

O conteúdo principal está centralizado em `src/data/profile.ts`.  
Os componentes de renderização estão em `src/components/site.ts`.

## Antes de publicar

Validar fotografia oficial, autorização da capa do livro, link oficial de LinkedIn, cargos atuais, publicações, links de media/podcast, FAQ clínica, páginas legais e domínio final.
